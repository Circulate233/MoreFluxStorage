# More Flux Storage

## 简介
事实上，这个 Mod 并未添加任何实际的“通量存储器”，而是作为 **通量网络 (Flux Networks)** 与其他 Mod 的功能性联动扩展。

该 Mod 添加了被称为“通量端口 (Flux Accessors)”的设备，为以下 Mod 提供了原生兼容接口：
- **龙之进化 (Draconic Evolution)**
- **通用机械 (Mekanism)**
- **应用通量 (Applied Flux / AE2)**

这些通量端口允许你通过通量网络直接访问对应 Mod 的存储系统，而无需使用传统的“输入点 (Plug)”和“接入点 (Point)”进行繁琐的中转操作。

## 支持版本
- 1.12.2 (Forge)
- 1.20.1 (Forge)
- 1.21.1 (NeoForge)

## English Contents
# More Flux Storage

## Overview
In fact, this mod does not add any actual "Flux Storage" units. Instead, it serves as a functional integration bridge between **Flux Networks** and other mods.

This mod introduces devices known as "Flux Accessors," providing native compatibility interfaces for:
- **Draconic Evolution**
- **Mekanism**
- **Applied Flux (AE2)**

These Flux Accessors allow you to directly access the storage systems of these mods via the Flux Network, eliminating the need for traditional "Plugs" and "Points" to bridge the connection.

## Supported Versions
- 1.12.2 (Forge)
- 1.20.1 (Forge)
- 1.21.1 (NeoForge)
